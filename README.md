# prisma-chat-client

## Getting Started

1. Install & run backend
```bash
git clone https://github.com/luvor/prisma-chat-backend
npm install
npx prisma migrate dev --name init
node index.js
```
2. Install & run frontend
```bash
git clone https://github.com/luvor/prisma-chat-client
npm install
npm run dev
```
